# EcoRappTech

Sito ufficiale di EcoRappTech – Consulenza e soluzioni per energia rinnovabile, CER/AUC, fotovoltaico, forniture energia/gas, monitoraggio.

**Servizi principali:**
- Consulenza tecnica, commerciale e amministrativa
- Forniture energia elettrica e gas (migliori tariffe)
- Impianti fotovoltaici, wall box, accumulo
- Attivazione Comunità Energetiche Rinnovabili (CER/AUC)
- Monitoraggio energetico con AI
- Analisi e controllo fatture energia
- Servizi specialistici (EGE, diagnosi, ISO 50001, bilanci sostenibilità)

**Target:** Privati · Aziende · Terzo Settore · Pubblica Amministrazione
